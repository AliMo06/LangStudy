Thank you for using our web-app LangStudy
Disclaimer: You must have Python 3.12.3 to run this program!
Here is a manual step-by-step on how to launch the web-app for the first time:
1. open up a git bash terminal
2. type in the following commands
    a. cd backend
    b. python3 -m venv venv source venv/bin/activate
    c. pip install -r requirements.txt
    d. python database.py
3. now your web-app is fully set-up
4. to run the web-app type in: python app.py
5. this will launch the application on the address 127.0.0.1:5000
6. you may now interact with the app on that web address
7. finally, to close the app, press CTRL + C in the git bash terminal
